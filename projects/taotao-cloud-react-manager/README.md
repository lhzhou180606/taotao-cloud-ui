
# taotao-cloud-backend

#### 1.后台管理系统简介

**taotao-cloud-backend** 
```
后台基于
Spring Boot 2.4.3 、
Spring Cloud 2020.0.1 、
Spring Cloud Alibaba 2.2.5.RELEASE 、
Spring Security 、
Spring Security Oauth2 、

前台基于
react 、
apollo 、
antd 、
graphql 、
lodash 、
less 、
sa 、

前后端分离的的RBAC权限管理系统，项目支持数据权限管理，
支持后端配置菜单动态路由,第三方社交登录, 努力做最简洁的后台管理系统
```

**taotao-cloud-backend** 后端管理源码
